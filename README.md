# WashPass NG - Premium Car Wash Subscription Platform

A high-fidelity visual prototype of a modern Nigerian car wash subscription web application designed to showcase the product concept to clients and investors.

## 🚀 Project Overview

WashPass NG is a premium car wash subscription service targeting the Nigerian market, with a focus on Lagos and Abuja. The platform offers three subscription tiers (Basic, Standard, Premium) and provides a seamless experience for users, partners, and administrators.

## 📱 Pages & Features

### 1. Landing Page (`index.html`)
- **Hero Section**: Eye-catching headline with subscription benefits
- **Pricing Cards**: Three-tier subscription plans (₦8,500 - ₦25,000/month)
- **How It Works**: 3-step process explanation
- **Locations Map**: Interactive map showing partner locations in Lagos and Abuja
- **Statistics**: User count, partner locations, and ratings

### 2. User Dashboard (`dashboard.html`)
- **Subscription Status**: Active plan details and remaining washes
- **QR Scanner**: Dynamic QR code for wash validation
- **Quick Stats**: Nearby locations, water saved, loyalty points, time saved
- **Recent Activity**: Wash history with points earned
- **Vehicle Management**: Multiple vehicle support
- **Location Finder**: Nearby partner locations with navigation

### 3. Partner Dashboard (`partner-dashboard.html`)
- **Real-time Queue Management**: Active wash tracking
- **Earnings Summary**: Daily/weekly/monthly revenue tracking
- **Wash History**: Completed services with duration and earnings
- **Performance Analytics**: Charts and key metrics
- **QR Scanner**: Customer code validation
- **Status Management**: Online/offline toggle

### 4. Admin Panel (`admin-panel.html`)
- **System Overview**: Key metrics and KPIs
- **Analytics Dashboard**: Revenue trends, user distribution, service popularity
- **User Management**: Search, view, edit, and suspend users
- **System Health**: Real-time monitoring of servers, database, and APIs
- **Activity Logs**: System events and notifications
- **Report Generation**: Export functionality for analytics
